# python-calendar

Консольный календарь на Python. Учебный проект для отработки базовых навыков.

## Что умеет
- Функция, которая запрашивает у пользователя год и повторяет запрос, пока ввод не станет корректным.
- Логика, разбитая на отдельные функции (ввод, проверка, основной цикл).
- Обработка простых краевых случаев (некорректный ввод).
- Понимает висакостный год или нет, и на какой день недели падает первое января, от чего строит остальные дни недели на год.

## Чему научился при создании
- Разбивать логику на отдельные функции.
- Работать с циклами и условиями.
- Обрабатывать граничные случаи
- Находить готовые формулы (для проверки на какой день недели падает первое января)
- Писать и вызывать функции с аргументами.
- Организовывать цикл для повторного запроса данных.
- Разделять логику на понятные блоки, чтобы код было легче читать и менять.
